# Customer Churn Prediction (Lab 1 & Lab 2)

## Overview
This project focuses on predicting **customer churn** using machine learning techniques.  
Customer churn refers to customers who stop using a company's service. The project analyzes customer data to identify patterns and predict whether a customer will churn or not.

## Dataset
The dataset contains telecom customer information such as:
- Gender
- Tenure
- Internet Service
- Contract Type
- Payment Method
- Monthly Charges
- Total Charges
- Churn (Target Variable)

## Lab Work

### Lab 1 – Data Analysis
- Imported and explored the dataset
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Visualization of churn patterns

### Lab 2 – Machine Learning
- Feature selection
- Train-test split
- Model training
- Model evaluation for churn prediction
